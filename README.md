# Lerna-Nuxt

Lerna-Nuxt is a Lerna Mono Repository for shared Vue Components/Libraries alongside a Nuxt Application

## Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd nuxt-app
$ yarn install
```

### Run Storybook

```sh
$ yarn storybook
```

### Run Nuxt Development Server

```sh
$ cd nuxtapp
$ yarn dev
```
